
# Duplicity

An Oxygen Not Included duplicant editor.

# Features

- Edit Duplicants
    - Name
    - Gender
    - Voice
    - Size (width and height)
    - Health Status (healthy, critical, incapacitated, invulnurable, ...)
    - Skills (level and experience)
    - Traits
    - Status Effects
